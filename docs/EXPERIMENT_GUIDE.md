# Experiment Guide

Planned experiments:

- Evidence ablation.
- Execution budget curve.
- Weak-test robustness.
- Marginal evidence contribution under each execution budget.
- Evidence complementarity analysis.
- Evidence-grounded failure explanation.
- Failure explanation.
- Language generalization.
- Static-only versus execution-only versus evidence fusion.

Every experiment must produce JSON, CSV, tables, and figures, and must update progress files.
