# Contributing

EviCode prioritizes reproducibility and auditability.

Before contributing:

1. Install dependencies from `requirements-dev.txt`.
2. Run `python -m pytest`.
3. Keep all scripts resume-safe.
4. Update `requirements.txt`, `REPRODUCIBILITY.md`, and `PROJECT_STATUS.md` when behavior changes.

Do not commit large datasets, model checkpoints, or generated experiment outputs unless explicitly intended for a release artifact.
